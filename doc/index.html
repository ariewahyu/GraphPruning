Package MLF

source code

This packaged contains a Python implementation of the Marginal Likelihood Filter for weighted networks (graphs). This filter works by computing a statistical significance measure for each weighted edge of the graph and thresholding based on this measure. This significance measure can be compouted for directed or undirected weighted graphs. If you use this filter in your research, please scite the following paper:

Navid Dianati, Unwinding the hairball graph: pruning algorithms for weighted complex networks, arXiv:1503.04085 [physics.soc-ph]

Requires:

  • igraph
